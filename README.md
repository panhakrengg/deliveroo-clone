# deliveroo-clone

### Installation:

1. Clone this project:

  ```bash
  https://github.com/panhakrengg/deliveroo-clone.git
  ```

2. Install dependencies:

  ```bash
  npm install
  ```
    
3. Run Enviroment:
  
  ```bash
  expo start
  ```
  
#Sanity:
